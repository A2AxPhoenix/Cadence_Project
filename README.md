BUILD INSTRUCTIONS:

Starting from the root directory, perform the following commands (You must also provide your own database!)
1. npm i
2. cd frontend
3. npm i
4. npm run build
5. cd ..
6. nodemon server.js

If done properly, this should start the localhost server on http://localhost:5501.
